# Countify-React
A React counter app to increase, decrease, reset, or manually set a number. Built with React and Tailwind CSS.

# Features

Increment and decrement counter (0–100)
Reset counter to 0
Enter a number manually

# Tech Stack

React – for building the UI
Tailwind CSS – for styling

# Live Demo
[View on Netlify](https://countify-react.netlify.app/)

# Installation
Install dependencies:
npm install

Start the app:
npm start

# Usage
Click ✚ to increase, ━ to decrease, 🗘 to reset

Enter a number in the input box to set counter manually
